# Portfolio

- SQL
- Предобработка данных

- Теория вероятностей
- Принятие решений в бизнесе (А/В-тестирование)
- Работа с гипотезами (Приоритезация и проверка гипотез)
- Визуализация данных (Matplotlib, Plotly, Seaborn, Folium)
- Построение дашбордов (Tableau)
